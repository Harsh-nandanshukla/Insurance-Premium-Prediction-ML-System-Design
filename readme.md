# Insurance Premium Prediction – Production ML System Design

A production-oriented Machine Learning inference system built using FastAPI and deployed using Docker and AWS EC2.

---

## 🚀 Overview

This project predicts the Insurance Premium Category (Low / Medium / High) based on user attributes such as age, BMI, income, occupation, and lifestyle risk.

The primary focus of this project is:
- Production-grade ML API design
- Modular backend architecture
- Containerized deployment workflow

---



---

## 🛠️ Tech Stack

- Python
- FastAPI
- Pydantic
- Scikit-learn
- Docker
- AWS EC2

---


---

## ☁️ Deployment

The application is containerized using Docker and deployed on AWS EC2, demonstrating an end-to-end production ML deployment pipeline.

---

## 📌 Purpose

This project emphasizes ML system design, structured validation, feature engineering at inference time, and real-world deployment practices.
